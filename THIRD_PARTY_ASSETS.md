# Third-Party Assets

No external runtime art, model, texture, audio, or icon assets are shipped.

Visuals are procedural low-poly geometry generated in source code. UI icons come from the npm package `lucide-react` under its package license and are used as library components, not bundled copied artwork.

If future assets are added, each entry must include title, creator, source, license, required attribution, modifications, file paths, and reviewer status.
